# Release Notes

## v1.0.0

Initial open source release of Claude Class.

### Included

- class and student management
- weekly and monthly check-in records
- built-in and custom evaluation dimensions
- dashboard metrics, rankings, and trends
- recycle bin workflow
- mobile-friendly PNG export
- local-first browser storage with no backend requirement

### Polished before release

- fixed dashboard double-counting between weekly and monthly analytics
- improved trend display so mixed period granularities are not shown together
- made top navigation accessible as real links
- improved export modal scrolling behavior
- aligned exported PNG output with the on-screen preview
- cleaned repository files for public open source publishing
