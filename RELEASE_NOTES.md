# 更新说明 / Release Notes

## v1.0.0 首次开源发布 / First Open Source Release

Claude Class 的首次开源发布版本。  
The first open source release of Claude Class.

### 已包含能力 / Included

- 班级与学生管理 / Class and student management
- 周记录与月记录 / Weekly and monthly check-in records
- 内置评价维度与自定义维度 / Built-in and custom evaluation dimensions
- 看板统计、排名与趋势 / Dashboard metrics, rankings, and trends
- 回收站恢复流程 / Recycle bin workflow
- 适合手机查看的 PNG 图片导出 / Mobile-friendly PNG export
- 无后端依赖的本地优先浏览器存储 / Local-first browser storage with no backend requirement

### 发布前已补齐优化 / Polished Before Release

- 修复看板里周/月统计口径重复累计的问题 / Fixed dashboard double-counting between weekly and monthly analytics
- 优化趋势展示，避免不同周期粒度混在同一条线上 / Improved trend display so mixed period granularities are not shown together
- 顶部导航改为真正可访问的链接 / Made top navigation accessible as real links
- 改善导出弹窗的滚动体验 / Improved export modal scrolling behavior
- 让导出的 PNG 与页面预览保持一致 / Aligned exported PNG output with the on-screen preview
- 清理仓库文件，使其适合公开开源发布 / Cleaned repository files for public open source publishing
